# **YUKI HIGURASHI オフィシャルサイト 仕様書**

**バージョン:** 4.0

**作成日:** 2025/12/08
**更新日:** 2026/02/03

**対象サイト:** YUKI HIGURASHI Official Website

**リポジトリ:** https://github.com/name190401/yuki-website

## **1\. サイト概要 (Overview)**

### **1.1 目的**

* 「日暮 有希」および「株式会社ANGELICA」のブランディング強化。
* Instagram (@yukisroom2025) と連携し、信頼感（静）と親しみやすさ（動）を両立させる。
* サービス（カウンセリング、スタイリング）の認知拡大と予約・問い合わせへの導線確保。

### **1.2 ターゲット**

* 自分らしいライフスタイルを模索している女性。
* メンタルケアや自己実現に関心がある層。
* Instagramのフォロワーおよび検索流入ユーザー。

### **1.3 コンセプト**

**テーマ:** 「人に寄り添い心の支えになる、明るくて元気を与える存在」

**キーワード:** 大人ピンク、余白、品格、静寂、信頼、あたたかさ、可愛い、リッチ。

男性的な「重厚感」ではなく、女性的な「しなやかさ・凛とした芯の強さ」を表現する。

## **2\. デザイン仕様 (Design System)**

### **2.1 カラーパレット**

サイト全体を通して、落ち着いた「大人ピンク」をアクセントに使用し、ベースは白と薄いグレーで清潔感を保つ。

| 役割 | 色名 | カラーコード | 用途 |
| :---- | :---- | :---- | :---- |
| **Main Text** | Charcoal | \#4a4a4a | 本文、見出し（柔らかい黒） |
| **Sub Text** | Light Gray | \#888888 | 英字サブタイトル、補足情報 |
| **Accent** | Dusty Pink | \#b78d95 | リンクホバー、装飾線、強調箇所 |
| **Background** | Snow White | \#ffffff | メイン背景色 |
| **Sub BG** | Pale Pink | \#fdf8f8 | モバイルメニュー背景、Contact背景、ローディング画面 |
| **Candy Pink** | Candy Pink | \#ff6b9d | レインボーグローエフェクト |
| **Coral** | Coral | \#ff8a80 | レインボーグローエフェクト |
| **Lavender** | Lavender | \#e1bee7 | レインボーグローエフェクト |
| **Rose Gold** | Rose Gold | \#f8bbd9 | レインボーグローエフェクト |

### **2.2 タイポグラフィ (Fonts)**

Google Fontsを使用し、和文と欧文を使い分けることで「和モダン」な雰囲気を演出。

* **和文 (Japanese):** [Shippori Mincho](https://fonts.google.com/specimen/Shippori+Mincho)
  * 明朝体。繊細で可読性が高く、品格を感じさせる。
* **欧文 (English):** [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)
  * セリフ体。クラシックでエレガントな印象。見出しやナビゲーションに使用。

### **2.3 レイアウト方針**

* **レスポンシブデザイン:** PC（1200px max）、タブレット、スマホ（768px以下）に対応。
* **余白 (White Space):** セクション間の余白を大きく取り（padding: 120px）、ゆったりとした世界観を作る。
* **非対称性 (Asymmetry):** 画像とテキストを少しずらして配置するなど、動きのあるレイアウトを採用。
* **グラスモーフィズム:** サービスカード等に半透明ブラー効果を適用し、モダンで高級感のあるデザイン。

## **3\. ページ構成・コンテンツ (Sitemap)**

マルチページ構成。各ページは独立したHTMLファイル。

* **Logo:** 左上に配置。サイトタイトルを表示。シンプルなテキストロゴ。ホバー時に色変化。
* **PC Menu:** 右上に配置。About, Gallery, Service, Contact へのリンク。ホバー時にグラデーション下線アニメーション。
* **Mobile Menu:** ハンバーガーメニュー（三本線）。クリックでドロワーメニューが右からスライドイン。

## **4\. 機能・技術仕様 (Technical Spec)**

### **4.1 使用技術**

* **HTML5:** セマンティックなマークアップ。
* **CSS3:** Flexbox, Gridレイアウト, CSS Variables（変数）, Media Queries, Keyframe Animations, backdrop-filter（グラスモーフィズム）。
* **JavaScript (Vanilla JS):** 軽量なインタラクション実装。
* **Lenis:** スムーススクロールライブラリ（CDN経由）。

### **4.2 実装機能**

1. **オープニングローディング:**
   * ページ読み込み時にピンク背景のカーテンを表示。
   * ロゴがフェードイン後、カーテンが上にスライドアウト。

2. **スクロールフェードイン (Scroll Animation):**
   * IntersectionObserver APIを使用。
   * 要素が画面内に入ると、下からふわっと浮き上がるクラス（.fade-in-up）を付与。

3. **スムーススクロール (Lenis):**
   * ページ全体の滑らかなスクロール体験。
   * イージング関数でリッチな動きを実現。

4. **桜の花びらエフェクト:**
   * ページ全体にピンクの花びらがふわふわと舞い落ちる。
   * SVGグラデーションで繊細な花びらを表現。
   * ソフトシャドウ付きで上品な印象。
   * スマホでは生成頻度を下げてパフォーマンス最適化。

5. **🦋 蝶々エフェクト:**
   * 画面を優雅に飛び回るピンク〜パープルの蝶々。
   * 羽ばたきアニメーション付き。
   * スマホでは生成頻度を下げてパフォーマンス最適化。

6. **💕 フローティングハートエフェクト:**
   * 画面下部から浮かび上がるカラフルなハート。
   * キャンディピンク系のグラデーション。
   * ふわふわと漂いながら上昇。

7. **🫧 ハートバブルエフェクト:**
   * シャボン玉のように浮かぶハート型バブル。
   * 虹色の反射効果付き。
   * ゆらゆら揺れながら上昇。

8. **キラキラパーティクル:**
   * 画面にランダムで小さな光の粒が瞬く。
   * 白〜ピンクのグラデーション。

9. **✨ マジックスターダストカーソル:**
   * マウスに追従してキラキラの星屑が発生。
   * ランダムな色（ピンク、ゴールド、ラベンダー）。
   * スマホでは自動的に無効化。

10. **カーソルトレイルエフェクト:**
    * マウスに追従してピンクの光跡が残る。
    * 多層グラデーション＆ソフトライト合成で自然な馴染み。
    * カーソル周囲にふんわり光るグロー。
    * スマホでは自動的に無効化。

11. **リップルクリックエフェクト:**
    * ボタン・リンクをクリックすると波紋が広がる。
    * 中心が白く外側がピンクのグラデーション。

12. **ふわっとバウンスエフェクト:**
    * 各インタラクティブ要素にホバー時の浮遊感。
    * cubic-bezier(0.34, 1.56, 0.64, 1) で弾むようなイージング。
    * 対象: サービスカード、ギャラリー画像、ボタン、ナビリンク、SNSアイコン、ロゴ。



14. **ロゴデザイン:**
    * シンプルなテキストロゴ。ホバー時に色変化。

15. **⌨️ タイピングアニメーション:**
    * ヒーローセクションで「ゆきの部屋」がタイプライター風に表示。
    * カーソル点滅エフェクト付き。

16. **🪟 グラスモーフィズム (Hero):**
    * ヒーローセクションのコンテンツボックスに半透明ブラー効果（backdrop-filter: blur）を適用。

17. **ハンバーガーメニュー (Mobile Navigation):**
    * JavaScriptでクラスの着脱（.active）を制御。
    * アイコンのアニメーション（×印への変形）を含む。
    * メニューリンククリック時に自動で閉じる仕様。

18. **外部リソース:**
    * アイコン: FontAwesome CDN
    * スムーススクロール: Lenis CDN

### **4.3 ブラウザ互換性**

* Chrome, Safari, Edge, Firefox の最新版で動作確認。
* Mobile Safari (iOS), Mobile Chrome (Android) 対応。

### **4.4 パフォーマンス最適化**

* スマホでは桜・蝶々・ハートエフェクトの生成頻度を下げる。
* スマホではカーソルトレイル・マジックスターダストを無効化。
* アニメーション終了後にDOM要素を自動削除してメモリリーク防止。

## **6\. 運用・更新について**

* **画像の差し替え:** img タグの src 属性を、実際の写真ファイルのパスまたはURLに変更することで行う。
* **リンクの変更:** Instagram以外のリンク先（予約フォームなど）ができた場合は、a タグの href 属性を変更する。
* **エフェクトの調整:** main.js の各関数内の数値（間隔、サイズ、速度など）を変更することでカスタマイズ可能。
